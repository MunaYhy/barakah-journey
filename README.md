# 🌿 Barakah Journey

A 90-day postpartum wellness tracker for Muslim women.

Built with Next.js 14 + Tailwind CSS + Recharts.

## Features
- Daily habit & prayer tracker
- Mood & emotional check-in
- Breastfeeding tracker
- Self-care routine
- Weight & measurements milestones
- Weekly review with star rating
- Rewards system
- Progress report with charts
- Dark mode
- Mobile (bottom nav) + Desktop (sidebar) layout
